# Cosentus AI - iframe Integration

**Copy-paste these codes into your website. That's it!**

---

## üí¨ Chat Assistant

**AI-powered chat for revenue cycle management questions**

```html
<iframe 
  src="https://cosentusai.vercel.app/embed/chat" 
  width="100%" 
  height="600" 
  style="border: none;">
</iframe>
```

**What you get:**
- Search bar with 7 pre-loaded RCM questions
- Expands to full chat when clicked
- Clean, minimal design
- Mobile-friendly

---

## üé§ All Voice Agents (Recommended)

**All 8 AI voice agents in one iframe - users can pick which one to talk to**

```html
<iframe 
  src="https://cosentusai.vercel.app/embed/voice/all" 
  width="100%" 
  height="800" 
  style="border: none;" 
  allow="microphone">
</iframe>
```

**What you get:**
- Grid of 8 agent cards
- Click any card to expand and start talking
- Each agent specializes in different tasks:
  - **Cindy** - Patient billing & payments
  - **Chris** - Insurance claim follow-ups
  - **Cassidy** - Anesthesia cost estimates
  - **Courtney** - Appointment scheduling
  - **Chloe** - General customer service
  - **Cara** - Insurance eligibility verification
  - **Carly** - Prior authorization tracking
  - **Carson** - Payment reconciliation

## üîß Sizing Tips

### Full-width (recommended):
```html
<iframe src="..." width="100%" height="800" style="border: none;"></iframe>
```

### Fixed width:
```html
<iframe src="..." width="1200px" height="800" style="border: none;"></iframe>
```

### Full-page:
```html
<iframe src="..." width="100%" height="100vh" style="border: none;"></iframe>
```
